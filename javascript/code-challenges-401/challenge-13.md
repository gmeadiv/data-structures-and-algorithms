# Stack/Queue Brackets

## Feature tasks

- Write a function called validate brackets
- Arguments: string
- Return: boolean
  - representing whether or not the brackets in the string are balanced

### There are 3 types of brackets

1. Round Brackets : ()

2. Square Brackets : []

3. Curly Brackets : {}

### Examples

- {} = true

- {}(){} = true

- ()[[EXTRA SCHARACTERS]] = true

- [ ({}] = false

- (]( = false
